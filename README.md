# EDYODA Assignment

EDYODA Clone using Vite React.

## Tech Stack

**Tech:** Vite React

**Packages:** react-router-dom

## Run Locally

Clone the project

```bash
  git clone https://github.com/adi-4444/EdYoda-Assignment
```

Go to the project directory

```bash
  cd edyoda-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
